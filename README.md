# EasyUi
Is a Rails gem that provides a set of pre-built UI components for your Rails application.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "easy_ui"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install easy_ui
```
